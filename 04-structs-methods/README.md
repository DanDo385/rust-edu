# Project 04 - Structs and Methods

## What You're Building

Learn how to create custom types with structs and add behavior with methods. Build a Rectangle type with various operations.

## The Exercises

1. **Rectangle::new**: Create a rectangle with validation
2. **area**: Calculate rectangle area
3. **perimeter**: Calculate rectangle perimeter
4. **can_fit**: Check if another rectangle fits inside
5. **scale**: Create a scaled copy

## How to Run

```bash
cargo test -p structs-methods
cargo run -p structs-methods
```

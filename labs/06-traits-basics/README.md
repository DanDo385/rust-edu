# Project 06 - Traits Basics

## What You're Building

Learn Rust's trait system (similar to interfaces). Create shapes that implement common behavior.

## Exercises

1. **Shape trait**: Define a trait for geometric shapes
2. **Circle, Rectangle**: Implement Shape for different types
3. **total_area**: Calculate total area of multiple shapes
4. **largest_shape**: Find shape with largest area

## How to Run

```bash
cargo test -p traits-basics
cargo run -p traits-basics
```

# Project 05 - Error Handling

## What You're Building

Learn Rust's error handling with Result and Option types. Build functions that parse, validate, and handle errors gracefully.

## Exercises

1. **parse_number**: Parse string to number with error handling
2. **divide**: Safe division returning Result
3. **read_first_line**: Read file's first line
4. **validate_email**: Check if string is valid email format

## How to Run

```bash
cargo test -p error-handling
cargo run -p error-handling
```
